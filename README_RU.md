# SimpleIDE

Легковесная среда разработки (IDE) для программирования на Python и использования веб-технологий.

## Возможности

SimpleIDE — это настольный редактор кода, работающий в отдельном окне. Возможности:

- **Управление проектами** — создание, открытие и инициализация проектов с помощью `project.json`
- **Файловый менеджер** — просмотр файлов и папок с поддержкой перетаскивания (drag-and-drop)
- **Редактор кода** — подсветка синтаксиса для более чем 40 языков (Python, JavaScript, C, C++, Java, Go, Rust, HTML, CSS и др.)
- **Запуск кода** — непосредственный запуск файлов Python и JavaScript
- **Встроенная консоль** — просмотр вывода и ввод команд
- **Предпросмотр HTML** — просмотр HTML-файлов в реальном времени
- **Вкладки** — возможность открывать несколько файлов одновременно
- **Поддержка языков** — интерфейс на русском и английском языках
- **Настраиваемость** — выбор темы, размера шрифта и фонового изображения

## Почему эта IDE — легковесная

- Создана на **Python + Eel** — использует системный браузер для интерфейса, без тяжелых фреймворков
- **Без Electron** — не включает в себя Chromium (использует уже установленный у вас Chrome или Edge)
- **Минимум зависимостей** — только `eel` и `watchdog`
- **Чистый JavaScript (Vanilla JS)** — без React, Vue или Angular
- **Компактность** — EXE-файл весит ~15 МБ, потребляет ~50 МБ оперативной памяти
- **Быстрый запуск** — загружается за 1–2 секунды
- **Модульный код** — чистая архитектура, легко поддается модификации

## Установка

### Скачать EXE (Windows)

1. Скачайте `SimpleIDE.exe` из раздела [Releases](https://github.com/teeqly/SimpleIDE/releases)
2. Запустите файл (требуется наличие установленного браузера).

### Из исходного кода

```bash
git clone https://github.com/teeqly/SimpleIDE.git
cd SimpleIDE
pip install -r requirements.txt
python app.py
```

### Сборка из исходного кода

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --add-data "src;src" --add-data "others;others" --name SimpleIDE app.py
```

## Требования

- Windows, Linux или macOS
- Python 3.8+ (для установки из исходного кода)
- Google Chrome или Microsoft Edge
